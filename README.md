# UAV-Machine-Learning
This project trains an unmanned aeirial vehicle to traverse through a 3D environment using reinforcement learning.

This projected was presented to a panel of professors for the STEM Fellowship (https://stemfellowship.org/) at York University and was awarded second place in the Computer Sceince, Engineering, Mathematics, and Physics Category.
The project poster is also included in this repo for more info on how this project works.
